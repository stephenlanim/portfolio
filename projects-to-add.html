React-Based Season Detector

This is a simple web app that detects whether the user is located in the northern or southern hemisphere and then renders a screen for either warm or cold weather, depending on the current season in your hemisphere.

You must allow the app to access your location, otherwise you will either see a spinner screen, or you will see an error message. If you did not see a spinner and only saw an error, your web browser automatically blocked the location request.

If you would like to play around with this app:

  • Open up developer tools.
  • Find the Sensors menu
  • Find the Geolocation dropdownmenu.
  • Select a location in a different hemisphere from your own.
  • Hint: If you have a positive number for latitude, select a location with a negative latitude. If you have a negative latitude, select a location with a positive latitude.
  • Reload the page and watch the screen change.
