About Me

I'm currently a front-end web developer, putting in my 10,000 hours toward mastering web app design and development.

I initially learned user interface design, HTML, CSS, and Bootstrap as part of my master of science program in instructional technology. But after completing some front-end web development projects for some local clients, I was hooked.

Deciding to go all the way with web development was one of the best decisions I've ever made. I wish I'd discovered my passion and talent for web development ages ago!
